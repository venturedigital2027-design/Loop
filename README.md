# Loop

Loop is a mobile-first web app for ultra-short playable games. The first MVP is a YouTube Shorts-style vertical feed where each card contains a simple playable loop.

## Current testable milestone

This version includes:

- Mobile-first dark interface
- Vertical snap scrolling feed
- Five seeded demo loops
- Right-side action bar
- Bottom navigation with Scroll and Build tabs
- Local scoring for demo games

## Run locally

```bash
npm install
npm run dev
```

Then open:

```text
http://localhost:3000
```

## Browser-only preview option

You can also try opening the public GitHub repo through StackBlitz:

```text
https://stackblitz.com/github/venturedigital2027-design/Loop
```

If StackBlitz asks for the start command, use:

```bash
npm run dev
```
